# go-webassemly

A small project for trying out webassemly with Go.

## directory simple

Print hello to browser console.

```
cd simple
bash build.sh
./server
```

Go to http://localhost:8000

## directory valuetoelement

Print hello to a web page.

```
cd valuetoelement
bash build.sh
./server
```

Go to http://localhost:8000
